# Conditions and Edge Cases

**Status:** 🔴 NOT YET DRAFTED

## Purpose

Covers mechanics that require special handling beyond the Finished Book's core eHP/eDPR framework.

Planned coverage:
- Legendary Actions and Lair Actions (initiative order integration)
- Legendary Resistance (resource tracking)
- Concentration (as a resource constraint on spell selection)
- Reaction economy (Opportunity Attacks, Shield, Counterspell)
- Full per-condition multiplier derivations: Restrained, Frightened, Prone, Blinded, Grappled, Poisoned, Exhaustion
- Complex spells: Polymorph, Banishment, Wall spells
